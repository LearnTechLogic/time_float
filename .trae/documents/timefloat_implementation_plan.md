# Timefloat 时光漂流局 - 项目实现计划

## 项目概述
开发一个基于 Flutter 和 Spring Boot 的时光漂流局网站，使用 MySQL 和 Redis 作为数据库。

## 整体架构
- **前端**：Flutter Web
- **后端**：Spring Boot
- **数据库**：MySQL (主数据库) + Redis (缓存)
- **部署**：待确定
