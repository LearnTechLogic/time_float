# 时光漂流瓶系统软件说明书

## 一、软件概述

### 1.1 软件名称
时光漂流瓶系统

### 1.2 软件版本号
V1.0.0

### 1.3 软件开发完成日期
2026年03月01日

### 1.4 软件简介
时光漂流瓶系统是一款基于移动互联网的社交互动应用，采用Flutter作为前端开发框架，Spring Boot作为后端服务框架，MySQL作为数据存储方案。系统通过"漂流瓶"这一经典互动形式，为用户提供一个匿名分享、随机交流的社交平台。用户可以发送包含文字内容的漂流瓶，也可以随机捞取他人发送的漂流瓶，并进行回复互动，实现跨越时空的社交体验。

## 二、软件功能

### 2.1 主要功能模块

#### 2.1.1 用户认证模块
- 用户注册：支持用户名和密码注册新用户
- 用户登录：已注册用户可通过用户名密码登录系统
- 身份认证：采用JWT（JSON Web Token）技术实现无状态身份认证
- Token管理：自动管理用户登录状态，支持安全退出

#### 2.1.2 漂流瓶发送模块
- 内容编辑：支持用户输入文字内容
- 漂流瓶投放：将编辑好的内容封装为漂流瓶投放到系统中
- 状态管理：记录漂流瓶的发送时间、发送者信息

#### 2.1.3 漂流瓶捞取模块
- 随机捞取：系统从待捞取的漂流瓶中随机选择一个返回给用户
- 状态更新：捞取成功后更新漂流瓶状态，标记为已捞取
- 捞取记录：记录捞取时间、捞取者信息

#### 2.1.4 漂流瓶管理模块
- 我的漂流瓶：查看自己发送的所有漂流瓶列表
- 捞取的漂流瓶：查看自己捞取的所有漂流瓶列表
- 漂流瓶详情：查看漂流瓶的详细内容和回复信息
- 删除功能：支持删除自己发送的漂流瓶

#### 2.1.5 回复互动模块
- 发送回复：针对捞取的漂流瓶发送回复内容
- 回复列表：查看漂流瓶的所有回复
- 对话交互：支持发送者和捞取者之间的多轮对话

### 2.2 功能流程图
```
用户注册/登录 → 主界面
    ↓
    ├─→ 发送漂流瓶 → 编辑内容 → 投放成功
    │
    ├─→ 捞取漂流瓶 → 随机获取 → 查看详情 → 发送回复
    │
    └─→ 我的漂流瓶 → 查看发送/捞取记录 → 管理漂流瓶
```

## 三、软件技术架构

### 3.1 总体架构
本系统采用前后端分离的B/S架构，前端使用Flutter实现跨平台移动应用，后端采用Spring Boot提供RESTful API服务，数据库使用MySQL进行数据持久化存储。

### 3.2 前端技术栈
- **开发框架**: Flutter 3.11.0+
- **编程语言**: Dart
- **状态管理**: Provider 6.1.1
- **网络请求**: http 1.1.0
- **本地存储**: shared_preferences 2.2.2, flutter_secure_storage 9.0.0
- **UI组件**: Material Design

### 3.3 后端技术栈
- **开发框架**: Spring Boot 3.2.10
- **编程语言**: Java 17
- **ORM框架**: MyBatis 3.0.4
- **数据库连接池**: HikariCP
- **身份认证**: JWT (JSON Web Token) 0.12.3
- **构建工具**: Maven 3.6+

### 3.4 数据库技术
- **数据库管理系统**: MySQL 8.0+
- **字符集**: utf8mb4
- **主要数据表**:
  - users（用户表）：存储用户基本信息
  - bottles（漂流瓶表）：存储漂流瓶内容和状态
  - replies（回复表）：存储漂流瓶的回复信息

### 3.5 系统架构图
```
┌─────────────────┐
│   Flutter 客户端  │
│  (Android/iOS)   │
└────────┬────────┘
         │ HTTP/REST
         ↓
┌─────────────────┐
│  Spring Boot    │
│   后端服务       │
└────────┬────────┘
         │ JDBC
         ↓
┌─────────────────┐
│   MySQL 数据库   │
└─────────────────┘
```

- 底部导航栏：包含"捞瓶子"、"扔瓶子"、"我的"三个标签页

- 内容编辑区域：文本输入框，支持输入文字内容

- 列表展示：以卡片形式展示漂流瓶列表

### 5.6 漂流瓶详情界面
- 漂流瓶内容展示区
- 回复列表区
- 回复输入框和发送按钮

## 六、软件创新点

1. **跨平台技术**: 采用Flutter框架实现一套代码多端运行，支持Android、iOS、Windows、macOS、Linux等多个平台。

2. **无状态身份认证**: 采用JWT技术实现无状态的身份认证机制，提高系统扩展性和性能。

3. **随机算法优化**: 设计高效的随机捞取算法，确保漂流瓶分配的公平性和随机性。

4. **前后端分离架构**: 采用RESTful API设计，实现前后端完全解耦，便于独立开发和部署。

5. **数据安全保障**: 密码加密存储、Token安全传输、CORS跨域配置等多重安全机制。

## 八、软件使用说明

### 8.1 用户注册
1. 打开应用，点击"注册"按钮
2. 输入用户名和密码
3. 再次确认密码
4. 点击"注册"完成账号创建

### 8.2 用户登录
1. 输入已注册的用户名和密码
2. 点击"登录"按钮进入系统

### 8.3 发送漂流瓶
1. 点击底部导航栏的"扔瓶子"
2. 在文本框中输入想要分享的内容
3. 点击"发送"按钮完成投放

### 8.4 捞取漂流瓶
1. 点击底部导航栏的"捞瓶子"
2. 点击"捞一个"按钮
3. 系统随机返回一个漂流瓶
4. 可查看详情并发送回复

### 8.5 查看我的漂流瓶
1. 点击底部导航栏的"我的"
2. 切换"我发送的"或"我捞到的"标签
3. 点击任意漂流瓶查看详情
