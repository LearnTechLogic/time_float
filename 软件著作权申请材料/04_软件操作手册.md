## 一、系统简介

时光漂流瓶系统是一款基于移动互联网的社交互动应用，用户可以通过发送、捞取漂流瓶的方式进行匿名社交互动，分享生活中的点滴，结识新朋友。

### 主要功能
- 用户注册与登录
- 发送漂流瓶
- 随机捞取漂流瓶
- 查看和管理我的漂流瓶
- 回复漂流瓶进行互动

## 三、安装部署

### 3.1 后端部署

#### 步骤一：数据库配置
1. 安装MySQL 8.0+数据库
2. 创建数据库：
```sql
CREATE DATABASE IF NOT EXISTS time_float DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```
3. 执行数据库初始化脚本 `schema.sql` 创建数据表

#### 步骤二：配置后端服务
1. 修改 `application.yml` 配置文件中的数据库连接信息：
```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/time_float?useUnicode=true&characterEncoding=utf-8&serverTimezone=Asia/Shanghai&useSSL=false
    username: your_username
    password: your_password
```

#### 步骤三：启动后端服务
```bash
cd time_float_backend
mvn clean spring-boot:run
```
后端服务将在 `http://localhost:8080` 启动

### 3.2 前端部署

#### 步骤一：安装依赖
```bash
cd time_float_frontend
flutter pub get
```

#### 步骤二：配置API地址
修改 `lib/services/api_service.dart` 中的 `baseUrl` 为实际的后端服务地址

#### 步骤三：运行应用
```bash
flutter run
```

## 四、功能操作说明

### 4.1 用户注册

#### 操作步骤
1. 启动应用，进入登录界面
2. 点击"注册"按钮，进入注册界面
3. 输入用户名（4-20个字符）
4. 输入密码（6-20个字符）
5. 再次输入密码进行确认
6. 点击"注册"按钮完成注册

#### 注意事项
- 用户名不能与已存在的用户名重复
- 密码长度至少6位
- 两次输入的密码必须一致

### 4.2 用户登录

#### 操作步骤
1. 在登录界面输入已注册的用户名
2. 输入密码
3. 点击"登录"按钮
4. 登录成功后进入主界面

#### 注意事项
- 请确保输入的用户名和密码正确
- 如忘记密码，请重新注册账号（当前版本暂不支持密码找回）

### 4.3 发送漂流瓶

#### 操作步骤
1. 登录后，点击底部导航栏的"扔瓶子"标签
2. 在文本输入框中输入想要分享的内容
3. 确认内容无误后，点击"发送"按钮
4. 系统提示"发送成功"后，漂流瓶已投入系统

#### 注意事项
- 内容不能为空
- 内容长度建议不超过500字
- 发送后无法修改内容，请谨慎发送

### 4.4 捞取漂流瓶

#### 操作步骤
1. 点击底部导航栏的"捞瓶子"标签
2. 点击"捞一个"按钮
3. 系统随机返回一个漂流瓶
4. 查看漂流瓶内容
5. 如想回复，点击"回复"按钮

#### 注意事项
- 每次只能捞取一个漂流瓶
- 漂流瓶被捞取后，其他人无法再捞取
- 如无漂流瓶可捞，系统会提示暂无漂流瓶

### 4.5 查看我的漂流瓶

#### 操作步骤
1. 点击底部导航栏的"我的"标签
2. 切换"我发送的"或"我捞到的"标签
3. 查看漂流瓶列表
4. 点击任意漂流瓶查看详情

#### 注意事项
- "我发送的"显示自己发送的所有漂流瓶
- "我捞到的"显示自己捞取的所有漂流瓶
- 可以查看漂流瓶的回复内容

### 4.6 回复漂流瓶

#### 操作步骤
1. 在漂流瓶详情页面，查看漂流瓶内容
2. 在回复输入框中输入回复内容
3. 点击"发送回复"按钮
4. 回复成功后会显示在回复列表中

#### 注意事项
- 回复内容不能为空
- 可以多次回复同一个漂流瓶
- 回复会通知对方查看

### 4.7 删除漂流瓶

#### 操作步骤
1. 在"我的"页面，找到要删除的漂流瓶
2. 长按或点击删除按钮（根据界面设计）
3. 确认删除操作
4. 漂流瓶删除成功

#### 注意事项
- 只能删除自己发送的漂流瓶
- 删除后无法恢复，请谨慎操作
- 删除后相关回复也会被删除

## 五、常见问题

### Q1：无法连接服务器？
A：请检查以下几点：
- 确认后端服务已正常启动
- 检查网络连接是否正常
- 确认前端配置的API地址正确

### Q2：注册时提示用户名已存在？
A：请更换其他用户名进行注册，用户名具有唯一性。

### Q3：捞不到漂流瓶？
A：可能是以下原因：
- 系统中暂无待捞取的漂流瓶
- 请稍后再试，或先发送一个漂流瓶

### Q4：忘记密码怎么办？
A：当前版本暂不支持密码找回功能，请重新注册账号。

### Q5：如何退出登录？
A：在"我的"页面，点击退出登录按钮即可。

### Q6：漂流瓶可以发送图片吗？
A：当前版本仅支持文字内容，图片功能将在后续版本中添加。
